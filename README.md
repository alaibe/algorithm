# Algorithm

A toy project to learn clojure 
and at the same time revised 
some algorithms:
 
* Insertion sort
* Selection sort
* Merge sort
* Quick sort
* Heap sort
* Bubble sort
* Tree sort
* Sequential search
* Binary search
* Dijkstra
* Linear regression
* RSA
* Bloom filter
* Miller-Rabin primality test

## Usage

Don't, it has no intent to be used,
You can probably find much better implementation elsewhere
